
Modificadores de Acesso :
    Public: Visibilidade para todos
    Private: visibilidade apenas para a classe
    Package Private: visibilidade apenas para o pacote
    Protected: Define o atributo visivel apenas para o propio pacote ou para os filhos

JAR -> O JAR serve para distribuir o codigo que foi compilado no caso classes, para outros projetos.
para importar um jar é preciso criar uma pasta padrão chamada libs e dentro dela colocar o arquvio compilado jar, após isso devemos clicar em BuildPath e add BuildPath.
O arquivo JAR nada mais é que um arquivo ZIP.
