Spring Framework o que é? -> spring framework é uma biblioteca que ajuda nas configurações do pom.xml, ele veio para facilitar as configurações do maven, junto com ele temos o Tomcat que abre um servidor local para facilitar o desenvolvimento de aplicações web

Tomcat -> É um servidor que é aberto na maquina local para desenvolvimento web.

DTO(Data Transfer Object) -> São classes que provê o que é necessário para determinado processo,
